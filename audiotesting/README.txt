This repository contains 2 scripts: manual_test.py and grade.py

manual_test.py instructions
1.	From the command line cd to the root directory and run manual_testing.py script
2.	Type manual_testing.py for help on command line arguments
a.	Note that the two scenarios should have the identical number of files in them 
3.	A testcases directory and answer_key.yml is then added to the root directory or specified output_directory


grade.py instructions
1.	Confirm that the anwer_key.yml and responses.xlsx are in the root directory or optional specified path
2.	From the command line, cd to the root directory and run grade.py
3.	See grade.py for information on command line arguments
4.	The test results are in test_results.yml in the root directory
